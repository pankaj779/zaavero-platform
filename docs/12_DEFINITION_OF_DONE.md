# Definition Of Done

A feature is considered complete only if ALL of the following are satisfied.

## Engineering

✓ Code Compiles

✓ TypeScript Passes

✓ ESLint Passes

✓ Production Build Passes

✓ No TODO comments

✓ No Placeholder Code

---

## Backend

✓ Validation Added

✓ Error Handling Added

✓ Logging Added

✓ Authorization Added

✓ Tests Passing

---

## Frontend

✓ Responsive

✓ Accessible

✓ Loading State

✓ Empty State

✓ Error State

✓ Mobile Friendly

---

## Database

✓ Migration Created

✓ Relationships Verified

✓ Constraints Verified

---

## Documentation

✓ Updated

---

## Git

✓ Clean Commit

✓ No Debug Code

---

Only after every checkbox is complete is a feature considered Done.
