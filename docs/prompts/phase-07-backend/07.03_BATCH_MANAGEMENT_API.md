=========================================================
BUSINESS RULES
=========================================================

A batch always belongs to exactly one course.

A teacher may own multiple batches.

Capacity (maxStudents) must always be greater than zero.

startDate must be before endDate.

Batch names are unique within a course.

Soft delete only.

Organization isolation required.

Teacher can modify only batches they own.

Admin can modify every batch inside their organization.
