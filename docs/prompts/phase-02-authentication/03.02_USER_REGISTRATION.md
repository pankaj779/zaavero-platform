# Business Rules

Email must be unique.

Phone must be unique if supplied.

Password must be hashed using Argon2.

Never store plaintext passwords.

Do not automatically verify email.

Do not log the password.

Do not generate JWT.

Do not log the user in.

---

# Organization

Every new user should automatically become a member of the default organization:

Graphology Academy

If the organization does not exist

Return a server error.

---
