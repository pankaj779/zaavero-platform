## API

Use the existing NestJS Calendar API.

Support

GET calendar-events

GET calendar-event

createCalendarEvent()

updateCalendarEvent()

deleteCalendarEvent()

Only the list endpoint is consumed by the current workspace.

Other CRUD methods should still exist for future dialogs.

---

## Mapping

Convert backend DTOs into the existing TeacherCalendarEventDto.

Keep every UI component unchanged.

Backend is the source of truth.

Only mapper placeholders may remain where the backend does not yet return fields.

Examples

meeting join URL

Google / Outlook sync status

teacher avatar

holiday artwork

reminder metadata

calendar color theme

Never invent backend properties.

---
