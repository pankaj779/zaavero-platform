# PRODUCT VISION

The Student Platform should feel like a premium SaaS application.

It should not feel like Moodle.

It should not feel like WordPress.

It should not feel like a coaching website.

It should feel closer to

• Notion

• Linear

• Stripe Dashboard

• Vercel Dashboard

Simple.

Fast.

Premium.

Calm.

Focused.
